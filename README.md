# Laboratorio Android 02 — Fundamentos de interfaces con Views y Kotlin

![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?style=flat&logo=kotlin&logoColor=white)
![Android](https://img.shields.io/badge/Android-3DDC84?style=flat&logo=android&logoColor=white)

Segundo laboratorio del curso de desarrollo Android. Profundiza en los fundamentos del diseño de interfaces con el sistema clásico de Views: layouts, componentes visuales, manejo de recursos, lectura de datos ingresados por el usuario y navegación básica entre pantallas.

## 📚 Temas cubiertos

- Views y ViewGroups fundamentales (`TextView`, `EditText`, `Button`, `ImageView`)
- Uso de recursos (`strings`, `colors`, `drawables`)
- Lectura de datos de usuario desde formularios
- Manejo básico de eventos (clics y entradas del usuario)
- Navegación simple entre Activities con `Intent`

## 🛠️ Tecnologías

| Tecnología | Uso |
|---|---|
| Kotlin | Lenguaje principal del laboratorio |
| Android SDK (Views/XML) | Construcción de interfaces y Activities |
| Android Studio | IDE de desarrollo y ejecución |

## ✅ Requisitos previos

Antes de ejecutar el proyecto necesitas tener instalado:

- [Android Studio](https://developer.android.com/studio) (versión reciente, incluye el Android SDK)
- JDK 17 (incluido con las versiones actuales de Android Studio)
- Un **emulador de Android** configurado desde el AVD Manager, **o** un **dispositivo físico** con la depuración USB activada

## 🚀 Instalación y ejecución

1. Clona el repositorio:

   ```bash
   git clone https://github.com/NinaDIV/Android-Lab02-Fundamentals.git
   cd Android-Lab02-Fundamentals
   ```

2. Abre el proyecto en Android Studio (`File → Open...` y selecciona la carpeta del repositorio).
3. Espera a que el IDE indexe el proyecto y descargue lo necesario.
4. Selecciona un emulador o conecta tu dispositivo físico.
5. Presiona **Run ▶** (o `Shift + F10`).

Al arrancar deberías ver la Activity principal del laboratorio en el emulador o dispositivo, con el formulario de ejemplo para probar los componentes.
