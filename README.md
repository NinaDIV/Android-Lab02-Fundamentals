# Android Lab 02 - Fundamentos de Android

Segundo laboratorio de Android. Profundiza en los fundamentos del diseno de interfaces: Views, ViewGroups y manejo de datos de usuario.

## Temas cubiertos
- Views y ViewGroups fundamentales (TextView, EditText, Button, ImageView)
- Uso de recursos (strings, colors, drawables)
- Lectura de datos de usuario desde formularios
- Navegacion simple entre Activities con Intent
